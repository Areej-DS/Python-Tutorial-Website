1. Introduction to Loops

Loops are used to execute a block of code repeatedly until a specified condition is met. In Python, there are two main types of loops: for loops and while loops.

2. The for Loop

Syntax

for variable in iterable:
    # Code to execute

Example: Iterating Over a List

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

Using the range() Function

# Printing numbers from 0 to 4
for i in range(5):
    print(i)

3. The while Loop

Syntax

while condition:
    # Code to execute

4. Loop Control Statements

break Statement

for i in range(10):
    if i == 5:
        break
    print(i)  # Will print numbers from 0 to 4

continue Statement

for i in range(5):
    if i == 2:
        continue
    print(i)  # Will print 0, 1, 3, 4

5. Summary

Loops are a fundamental part of programming, allowing you to execute code multiple times. Understanding for and while loops is essential for writing efficient code.
