Working with Numbers in Python

1. abs() - Absolute Value

Returns the absolute value of a number.

# Example: abs()
number = -15
print(abs(number))  # Output: 15

2. pow() - Power

Raises a number to a power.

# Example: pow()
print(pow(2, 3))  # Output: 8

3. sum() - Sum

Returns the sum of all items in an iterable.

# Example: sum()
print(sum([1, 2, 3, 4]))  # Output: 10

Working with Strings in Python

1. upper() - Convert to Uppercase

Converts a string to uppercase letters.

# Example: upper()
text = "hello"
print(text.upper())  # Output: HELLO

2. lower() - Convert to Lowercase

Converts a string to lowercase letters.

# Example: lower()
text = "HELLO"
print(text.lower())  # Output: hello

3. replace() - Replace Substring

Replaces parts of a string with another string.

# Example: replace()
text = "I love Python"
new_text = text.replace("love", "like")
print(new_text)  # Output: I like Python

4. split() - Split String

Splits a string into a list based on a delimiter.

# Example: split()
text = "apple,banana,cherry"
fruits = text.split(",")
print(fruits)  # Output: ['apple', 'banana', 'cherry']

Working with Lists in Python

1. append() - Add to a List

Adds an element to the end of a list.

# Example: append()
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # Output: [1, 2, 3, 4]

2. remove() - Remove from a List

Removes the first occurrence of an element from a list.

# Example: remove()
my_list = [1, 2, 3, 4]
my_list.remove(2)
print(my_list)  # Output: [1, 3, 4]

3. count() - Count Occurrences

Returns the number of occurrences of a specified value in the list.

# Example: count()
numbers = [1, 2, 3, 2, 1]
print(numbers.count(2))  # Output: 2

Working with Tuples in Python

1. tuple() - Create a Tuple

Converts a list into a tuple.

# Example: tuple()
my_list = [1, 2, 3]
my_tuple = tuple(my_list)
print(my_tuple)  # Output: (1, 2, 3)

2. min() and max() - Min and Max Values in Tuple

Returns the smallest or largest value from a tuple.

# Example: min() and max()
my_tuple = (5, 10, 3, 8, 15)
print(min(my_tuple))  # Output: 3
print(max(my_tuple))  # Output: 15

3. sorted() - Sort a Tuple

Returns a sorted list from the elements of a tuple.

# Example: sorted()
my_tuple = (3, 1, 4, 2)
sorted_tuple = sorted(my_tuple)
print(sorted_tuple)  # Output: [1, 2, 3, 4]
