The Python Data Types

Data Type Description & Example
String A string is a sequence of characters. In Python, strings are enclosed in quotes. 
# Example
my_string = "Hello World!"
Integer An integer is a whole number without a fractional part. 
# Example
my_integer = 7
Float A float is a number with a decimal point or in exponential form. 
# Example
my_float = 9.49
List A list is an ordered collection of items that can be of different types. 
# Example
my_list = [10, 20, 30, "apple", 40.5]
Dictionary A dictionary is a collection of key-value pairs. 
# Example
my_dict = {
    "name": "Areej",
    "age": 20,
    "city": "Makkah"
}
