Basic List Syntax

You can create a list by placing elements inside square brackets [], separated by commas:

my_list = [1, 2, 3, "Python", [4, 5]]

Common List Operations

1. Accessing List Elements

List elements are indexed starting from 0. You can access individual elements using the index:

my_list = ['a', 'b', 'c', 'd']
  print(my_list[0])  # Output: 'a'

2. List Slicing

You can access a portion of the list using slicing:

my_list = [1, 2, 3, 4, 5]
  print(my_list[1:4])  # Output: [2, 3, 4]

3. Modifying Lists

Since lists are mutable, you can change an element at a specific index:

my_list = [10, 20, 30, 40]
  my_list[2] = 100
  print(my_list)  # Output: [10, 20, 100, 40]

Common List Methods

1. Appending Elements

You can add elements to the end of a list using the append() method:

my_list = [1, 2, 3]
  my_list.append(4)
  print(my_list)  # Output: [1, 2, 3, 4]

2. Inserting Elements

The insert() method allows you to add an element at a specific position:

my_list = [1, 2, 3]
  my_list.insert(1, 100)
  print(my_list)  # Output: [1, 100, 2, 3]

3. Removing Elements

You can remove elements from a list using remove() or pop(). The remove() method removes the first occurrence of a value, while pop() removes an element at a specified index (or the last element by default).

my_list = [1, 2, 3, 2]
  my_list.remove(2)
  print(my_list)  # Output: [1, 3, 2]
  
  my_list.pop(1)
  print(my_list)  # Output: [1, 2]

4. List Length

Use the len() function to get the number of elements in a list:

my_list = [10, 20, 30]
  print(len(my_list))  # Output: 3

5. Sorting a List

You can sort a list in place using the sort() method:

my_list = [3, 1, 2]
  my_list.sort()
  print(my_list)  # Output: [1, 2, 3]

6. Reversing a List

The reverse() method reverses the elements of the list in place:

my_list = [1, 2, 3]
  my_list.reverse()
  print(my_list)  # Output: [3, 2, 1]

Advanced List Features

1. List Comprehensions

List comprehensions provide a concise way to create lists. They consist of brackets containing an expression followed by a for clause:

squares = [x**2 for x in range(5)]
  print(squares)  # Output: [0, 1, 4, 9, 16]

2. Nesting Lists

Lists can contain other lists, enabling you to create complex data structures:

nested_list = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
  print(nested_list[1][2])  # Output: 6
