1. Introduction to Arrays

An array is a data structure that can hold multiple values of the same type. In Python, while you can use lists as arrays, you can also use the array module or the numpy library for more efficient numerical operations.

2. Using the array Module

Creating an Array

import array

# Create an array of integers
arr = array.array('i', [1, 2, 3, 4, 5])
print(arr)

Common Types

  • 'i': Signed integer
  • 'f': Float
  • 'd': Double

Accessing Elements

print(arr[0])  # Access first element

Array Operations

# Append
arr.append(6)

# Insert
arr.insert(2, 10)  # Insert 10 at index 2

# Remove
arr.remove(3)  # Remove first occurrence of 3

# Slice
print(arr[1:4])  # Get elements from index 1 to 3

# Length
print(len(arr))

3. Using NumPy Arrays

Installation

pip install numpy

Creating NumPy Arrays

import numpy as np

# Create a 1D array
arr_1d = np.array([1, 2, 3, 4, 5])
print(arr_1d)

# Create a 2D array (matrix)
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr_2d)

Common Array Operations

# Shape
print(arr_2d.shape)

# Indexing
print(arr_2d[0, 1])  # Access element in first row, second column

# Slicing
print(arr_2d[0, :])  # Get first row
print(arr_2d[:, 1])  # Get second column

4. Array Manipulation

# Reshape
arr_reshaped = arr_1d.reshape((5, 1))

# Flatten
arr_flat = arr_2d.flatten()

# Concatenation
arr_concatenated = np.concatenate((arr_1d, arr_1d))

# Mathematical Operations
arr_sum = arr_1d + 2  # Add 2 to each element
arr_product = arr_1d * 2  # Multiply each element by 2

5. Practical Examples

Example 1: Basic Statistics

data = np.array([10, 20, 30, 40, 50])

mean = np.mean(data)
median = np.median(data)
std_dev = np.std(data)

print(f"Mean: {mean}, Median: {median}, Standard Deviation: {std_dev}")

Example 2: Matrix Operations

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Matrix addition
C = A + B

# Matrix multiplication
D = np.dot(A, B)

print("Matrix C:\n", C)
print("Matrix D:\n", D)

6. Summary

Arrays in Python can be created using the built-in array module or with NumPy for more advanced features. NumPy provides powerful capabilities for array manipulation, mathematical operations, and working with multi-dimensional data.
